Följande är en beskrivning av We Know IT:s arbetsprov i React Native.

a. Beskrivning:
Programmet återger invånarantalet för en stad genom att antingen söka efter specifika städer eller länders städer. 


b. Använda teknologier:
-Expo version 45.0.0
-React Native 0.68.2
-TypeScript 4.3.5
-React Navigation 6.0.10
-Redux version 4.2.0
-Node version 14.15.4
-Babel version 7.11.0

c. Instruktioner för att starta programmet
-Öppna expo.dev och följ instruktionerna för installation
-Nu kan du välja om du vill visa programmet på en simulator eller i telefonen
Ladda ner och installera Xcode om du använda simulator på en Mac, samma procedur för PC fast Android studio. För telefonanvändning behöver du hämta appen Expo Go.
-Försäkra dig om att du har projektmappen tillgänglig
-Öppna terminalen på din maskin och leta fram projektmappen i katalogen.
-Starta programmet genom att ange kommandot: expo start
